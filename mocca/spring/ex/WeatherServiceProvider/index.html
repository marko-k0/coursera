Weather Service App (P2 - Assignment 3)

Overview of the Weather Service App

The purpose of this "mini-project" assignment is to give you experience using Android to develop a complete Weather Service app using the Retrofit library, Android's Content Providers, and a SQLite database. Retrofit is used to obtain weather information from a Weather Service web service, and this data is then stored in a timeout cache that uses a database-backed Content Provider for storage. The user will provide a location as a String (such as "Nashville, TN"), and a WeatherData object will be returned to the WeatherActivity that can be used to display various current weather-related information for that location.
